Echo delay → coating thickness

How it works: A THz pulse reflects off the tablet surface (Echo 1) and again off the coating-core interface (Echo 2). The time delay Δt between echoes gives coating thickness: d = c · Δt / (2n). Drag the thickness selector to see the echoes move apart.

WBTM-Net architecture

Stage 1  Wavelet denoising — scale-adaptive threshold λ*(j) = σ √(2 ln N) · 2-j/2

Stage 2  Dispersive TMM — Sellmeier n(f) model, exact N-layer reflection

Stage 3  Bayesian MCMC — joint posterior p(d, n | signal), 95% CI output

Stage 4  Residual CNN — corrects physics model error, trains on ~50 samples
